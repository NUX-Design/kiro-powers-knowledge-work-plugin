# Close Management

Manage the month-end close process with task sequencing, dependencies, and status tracking.

This workflow assists with close execution planning but should still be governed by your organization’s accounting controls.

## Typical 5-Day Close

### Pre-Close

- send deadline reminders
- confirm cut-off procedures
- verify sub-systems are operating
- review open accrual candidates

### T+1

- finalize core cash and payroll activity
- run AP accruals
- post intercompany activity
- complete bank reconciliation
- run depreciation and prepaid amortization

### T+2

- complete revenue recognition
- post remaining accruals
- reconcile AR and AP subledgers
- post FX revaluation
- begin balance sheet reconciliations

### T+3

- finish balance sheet reconciliations
- post reconciliation adjustments
- complete intercompany reconciliation
- run preliminary trial balance and flux review

### T+4

- post tax and equity activity
- finalize draft statements
- complete detailed variance analysis
- perform management review

### T+5

- post final adjustments
- hard close and lock the period
- distribute the reporting package
- update forecasts
- run a retrospective

## Dependency Model

Group tasks by dependency level and identify the critical path from source entries through reconciliations, tax provision, draft financials, review, and hard close.

## Status Tracking

Use a status board with:

- task
- owner
- deadline
- status
- blocker
- notes

Recommended statuses:

- Not Started
- In Progress
- Complete
- Blocked
- At Risk

## Metrics To Track

- total close duration
- adjusting entries after soft close
- late tasks
- reconciliation exceptions
- post-close corrections

## Output

Produce:

- a close checklist or calendar
- dependency-sensitive sequencing
- a status dashboard template
- blocker and escalation guidance
