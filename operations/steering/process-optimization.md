# Process Optimization

Analyze an existing workflow, identify waste, and propose a higher-efficiency future state.

## Analysis Framework

### 1. Map Current State

- every step
- every handoff
- every decision point
- timing and waiting

### 2. Identify Waste

- waiting
- rework
- handoffs
- over-processing
- manual work

### 3. Design Future State

- remove unnecessary steps
- automate where possible
- reduce handoffs
- parallelize independent work
- add checkpoints instead of excessive gates

### 4. Measure Impact

- time saved
- error reduction
- cost savings
- employee experience improvement

## Output

Produce:

- current-state summary
- future-state recommendation
- before versus after comparison
- impact estimate
- implementation plan
