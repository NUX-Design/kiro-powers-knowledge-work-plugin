# Start And Orientation

Help a biological researcher get oriented with the available tools and workflows before starting work.

## Workflow

### 1. Welcome

Introduce the environment as a life-science research assistant spanning literature, analysis pipelines, and scientific strategy.

### 2. Check Connected MCP Servers

Survey available tool categories and group them by function:

- literature and research sources
- drug discovery and clinical sources
- visualization and AI biology tools

Report what is connected versus not yet configured.

### 3. Survey Available Workflows

List the main analysis paths:

- single-cell RNA QC
- scvi-tools
- Nextflow pipelines
- instrument data conversion to Allotrope
- scientific problem selection

### 4. Mention Optional Binary Additions

Call out that some genomics or tool-database capabilities may require separate binary MCP installation and are optional.

### 5. Ask What The Researcher Needs

Offer starting points such as:

- literature review
- sequencing data analysis
- drug discovery lookup
- data standardization
- research strategy
