# Performance Review

Use this workflow during review cycles to produce structured self-assessments, manager reviews, or calibration prep.

## Modes

- self-assessment
- manager review
- calibration prep

If the mode is not specified, ask which one is needed.

## Self-Assessment Mode

Generate a template covering:

- top accomplishments
- goals review
- growth areas
- challenges
- goals for next period
- feedback for the manager

## Manager Review Mode

Generate a review structure covering:

- overall rating
- performance summary
- strengths
- development areas
- goal achievement
- impact and contributions
- development plan
- compensation or promotion recommendation if relevant

## Calibration Mode

Generate a prep document covering:

- team overview
- rating distribution
- calibration discussion points
- promotion candidates
- compensation actions
- manager notes

## Connector-Aware Enhancements

If available:

- HRIS can prefill employee and historical review details
- project or work-tracking systems can supply evidence

## Output Rule

Feedback should be:

- specific
- behavior-based
- actionable
- evidence-backed

Avoid vague praise or personality labeling.
