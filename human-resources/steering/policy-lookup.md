# Policy Lookup

Use this workflow to answer handbook or policy questions in plain language with explicit source attribution.

## Use Cases

- PTO and leave questions
- benefits questions
- remote-work or travel-policy questions
- expense-policy questions
- conduct or procedure questions

## How to Answer

1. Search the connected knowledge base or the pasted handbook material.
2. Find the most relevant policy section.
3. Answer the question directly in plain language.
4. Quote or cite the exact source section.
5. Note any exceptions, gaps, or edge-case escalation points.

## Output Format

```markdown
## Policy: [Topic]

### Quick Answer
[direct answer]

### Details
[plain-language explanation]

### Exceptions / Special Cases
[exceptions]

### Who to Contact
[person or team]

### Source
[document and section]
```

## Guardrails

- Always cite the source.
- If no policy is found, say so clearly instead of guessing.
- For legal, tax, or compliance edge cases, direct the user to HR or legal review.
