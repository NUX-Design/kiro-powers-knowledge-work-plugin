# Recruiting Pipeline

Use this workflow to track candidate flow from sourcing through accepted offer, and to measure funnel health.

## Pipeline Stages

- sourced
- screen
- interview
- debrief
- offer
- accepted

## Metrics to Track

- pipeline velocity
- stage conversion rates
- source effectiveness
- offer acceptance rate
- time to fill

## Workflow

1. Pull candidate-stage information from the ATS if available, or accept a manually provided pipeline list.
2. Summarize candidate counts by stage.
3. Identify bottlenecks and stage drop-off.
4. Surface source and conversion insights.
5. Recommend next actions for the recruiting team.

## Output Guidance

Prefer clear stage tables and funnel commentary rather than long prose.

## Guardrails

- Do not overstate causality from limited pipeline data.
- If ATS data is missing, say the analysis is based on a manual snapshot.
