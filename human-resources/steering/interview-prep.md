# Interview Prep

Use this workflow to create a structured interview kit for a role.

## Core Principles

- structured and repeatable
- competency-based
- evidence-focused
- diverse-panel aware
- rubric-backed rather than gut-feel driven

## Workflow

1. Define 4 to 6 core competencies for the role.
2. Create a question bank by competency.
3. Add behavioral and situational questions.
4. Add useful follow-up probes.
5. Build a consistent scorecard.
6. Provide a debrief template and panel assignment guidance.

## Output

Produce:

- panel assignment guidance
- competency map
- question bank
- scorecard or rubric
- debrief template

## Guardrails

- Use the same core questions for all candidates in the same role when possible.
- Map each question to an explicit competency.
- Make scoring criteria clear enough that interviewers can calibrate.
