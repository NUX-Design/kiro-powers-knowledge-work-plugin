# Hiring

Use this workflow to build an end-to-end hiring packet for a small business: job post, interview guide, scoring rubric, and offer-letter template.

This steering file is derived from the source plugin’s `job-post-builder`.

## Scope

This workflow helps with:

- hiring brief intake
- market-informed job-post drafting
- stage-by-stage interview guide creation
- scoring rubric creation
- offer-letter template drafting
- optional DocuSign draft setup

It does not screen or rank applicants.

## Workflow

### 1. Gather role context

Collect:

- role title
- team or function
- key responsibilities
- must-have qualifications
- nice-to-have qualifications
- location and remote expectations
- compensation range if available
- interview process structure
- preferred offer-letter delivery path

If the user already provided most of this, confirm rather than re-asking.

### 2. Research baseline materials

Check for:

- existing internal job descriptions
- offer-letter templates
- interview guides
- a few live external postings for comparable roles

Use research to improve the output, not to overwrite the owner’s existing language unnecessarily.

### 3. Write the job post

If an existing template exists, preserve the structure and ask before adding missing sections. If not, create a market-aware post from scratch.

### 4. Build the interview guide

Organize by interview stage. Each stage should include:

- focus area
- behavioral questions
- follow-up probes
- scoring anchors

Avoid overlapping competencies across interviewers where possible.

### 5. Draft the offer letter

Use bracketed placeholders for missing specifics. Include:

- role and start details
- compensation placeholders if needed
- relevant legal or employment placeholders
- clear note that legal review is still required

### 6. Optional DocuSign draft setup

If requested:

- prepare the envelope
- add signer and message
- place signature fields
- save as draft

Do not send the envelope without explicit confirmation.

## Deliverables

Produce:

- job post document
- interview guide and rubric
- offer-letter template

If DocuSign is used, also return the draft-envelope status or link.

## Guardrails

- Never publish the job post automatically.
- Never send the DocuSign envelope automatically.
- Never invent compensation figures.
- Remind the user that legal and HR review still matter.
