# Performance Reporting

Turn marketing metrics into reports with trends, insights, and prioritized recommendations.

## Inputs

1. Report type
2. Time period
3. Data source
4. Comparison period
5. Stakeholder audience

If analytics tools are connected, pull platform data directly. Otherwise request pasted metrics, spreadsheet data, or summarized dashboard numbers.

## Report Structure

### 1. Executive Summary

- 2 to 3 sentence overview
- headline metric and direction
- one key win and one concern

### 2. Key Metrics Dashboard

```markdown
| Metric | This Period | Prior Period | Change | Target | Status |
|--------|------------|--------------|--------|--------|--------|
```

### 3. Trend Analysis

- notable movements
- inflection points
- seasonal or contextual effects
- benchmark comparison

### 4. What Worked

- top wins
- likely reasons
- ways to scale

### 5. What Needs Improvement

- weakest areas
- performance hypotheses
- recommended fixes

### 6. Insights

- patterns not obvious from top-line metrics
- audience or content behavior
- channel-specific observations

### 7. Recommendations

For each action include:

- what to do
- why
- expected impact
- effort
- priority

### 8. Next Period Focus

- top priorities
- experiments to run
- target metrics

## Metric Guidance

Use metrics appropriate to the report type, such as:

- campaign: reach, CTR, conversion, CPA, ROAS
- email: opens, clicks, unsubscribes, conversions
- social: reach, engagement, follower growth, clicks
- paid: spend, CPC, CPM, CPA, ROAS
- SEO: sessions, rankings, backlinks, indexed pages
- overall: MQLs, pipeline, CAC, sourced revenue
