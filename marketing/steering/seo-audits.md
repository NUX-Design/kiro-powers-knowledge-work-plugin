# SEO Audits

Audit a site or topic for keyword opportunities, on-page issues, content gaps, technical SEO health, and competitor comparisons.

## Inputs

Gather:

1. URL, domain, or topic
2. Audit type
3. Target keywords
4. Competitors when relevant

Default to a full site audit if audit type is not specified.

## Audit Areas

### Keyword Research

- primary keywords
- supporting keywords
- difficulty or competitiveness
- long-tail opportunities
- intent classification

### On-Page SEO

- title tags
- meta descriptions
- heading structure
- keyword placement
- internal linking
- alt text
- URL structure

### Content Gap Analysis

- missing topics
- stale content
- thin content
- missing formats
- funnel-stage gaps

### Technical SEO

- speed
- mobile experience
- structured data
- crawlability
- broken links
- HTTPS
- Core Web Vitals signals
- indexation risks

### Competitor SEO Comparison

- keyword overlap and gaps
- content depth
- backlink or authority signals
- SERP feature ownership
- technical differences

## Output

- executive summary
- keyword opportunity table
- on-page issues table
- content gap recommendations
- technical checklist
- competitor comparison summary
- prioritized action plan split into quick wins and strategic investments
