# Email Sequences

Design complete email flows with copy, timing, branching logic, exit conditions, and performance benchmarks.

## Inputs

Gather:

1. Sequence type
2. Goal
3. Audience and segmentation
4. Number of emails
5. Timing preferences
6. Brand voice
7. Offers, assets, or destination links

## Workflow

### 1. Sequence Strategy

Define:

- narrative arc
- user or buyer journey stage per email
- escalation logic
- success definition

### 2. Individual Email Design

For each email provide:

- 2 to 3 subject line options
- preview text
- purpose
- full body copy
- primary CTA
- timing
- segment or condition notes

### 3. Flow Logic

Specify:

- branching conditions
- exit conditions
- re-entry rules
- suppression rules

### 4. Performance Benchmarks

Track benchmarks by sequence type for:

- open rate
- click-through rate
- conversion rate
- unsubscribe rate

## Common Sequence Types

- onboarding
- lead nurture
- re-engagement
- win-back
- product launch
- event follow-up
- upgrade or upsell
- educational drip

## Output

Provide:

- sequence overview table
- full email drafts
- a text flow diagram
- branching logic notes
- A/B test suggestions
- metrics to track and review cadence
